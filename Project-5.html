Programming & Simulation

From Python game design to advanced MATLAB aerospace simulations : exploring numerical modelling and control.

Python - Interactive Game Development

As part of an academic project, I developed a complete interactive game in Python using object-oriented programming. The goal was to design a modular, maintainable, and efficient code structure with a graphical interface and basic physics interactions.

The project included player controls, event management, collision detection, and scoring systems, all implemented in Python. This experience strengthened my understanding of algorithms, classes, and game logic structuring.

MATLAB -Overbraiding Process Modelling

During my R&D internship at Airbus, I developed a MATLAB model for composite overbraiding on cylindrical and elliptical mandrels. The objective was to predict the local braiding angle and fiber path for different process parameters, and evaluate its applicability to existing industrial components.

  • Conducted an extensive literature review on overbraiding mechanisms and geometry prediction models.
  • Developed a simplified analytical model to estimate braiding angles along the mandrel perimeter.
  • Implemented more complex MATLAB simulations integrating mandrel geometry and fiber overlap constraints.
  • Analyzed model applicability to Airbus structures and defined a validation test plan for physical trials.

Simulation of overbraiding geometry on elliptical mandrel - MATLAB analytical model.

MATLAB - Missile Guidance Simulation

This project focused on the simulation of missile guidance laws using MATLAB and Simulink. The objective was to intercept a moving target by modelling both target motion and missile response under different control strategies.

  • Modelled pursuit, lead, and proportional navigation guidance laws.
  • Simulated dynamic target motion including oscillatory and non-uniform velocities.
  • Implemented a feedback control loop ensuring stability and minimizing miss distance.
  • Validated the system performance with a final impact deviation below 10 cm.

MATLAB/Simulink simulation of missile trajectory interception - proportional navigation control.

MATLAB - Hypersonic Flow

In this analytical project, I modelled shockwave formation and pressure evolution over the body of the Ariane 5 launcher during hypersonic flight regimes. Using MATLAB, I implemented both normal and oblique shock equations to study the aerodynamic loading and thermal effects on the launcher structure.

  • Implemented shock and expansion models based on compressible flow theory.
  • Computed post-shock Mach number, pressure, and temperature distributions along the vehicle body.
  • Identified detached shock behavior over the fairing and expansion waves near the interstage.
  • Validated trends with aerodynamic databases and analytical reference values.
